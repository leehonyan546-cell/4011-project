# AI-Based Real-Time Monitoring for PPE Compliance

## 📌 Project Overview
This project develops an automated safety monitoring system using **YOLOv11** to detect Personal Protective Equipment (PPE) violations in industrial environments. It specifically identifies missing helmets and gloves and sends instant visual alerts to supervisors via the **Telegram Bot API**.

## 🚀 Key Features
- **Real-Time Detection**: High-speed object detection using the latest YOLOv11 framework.
- **Automated Alerts**: Captures snapshots of safety violations (Class ID 7 for No-Helmet, ID 5 for No-Gloves).
- **Instant Notification**: Integrates with Telegram to provide supervisors with immediate visual evidence.
- **Hardware Optimized**: Supports Apple Silicon (MPS) acceleration for efficient local processing.
